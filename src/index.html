About Kayla

From Educator to
Web Developer

I enjoy a challenge - solving problems, bringing order to chaos, creating something useful to others. I became an educator for those reasons, and those are the reasons I am now a web developer.

In 2017, I began dabbling in code. To my surprise, I loved it. I loved the order and structure of HTML and the artistry and magic of CSS. As I continued to delve into coding, I found satisfaction in using the logic of Javascript to make things happen.

At the end 2020, I began studying to be a web developer. I worked through Codecademy’s front end developer course, supplementing along the way with resources from Kevin Powell, Traversey Media, Coding Train and Stephen Grider. As is often the case, I learned the most from the things I built myself. This portfolio showcases a few.

I’m excited to continue to learn and grow as a software developer as I take my first steps into this industry.

Climbing Company’s Internet Debut

I designed and built this website for a startup climbing company: Pisgah Climbing School of North Carolina. This is a responsive, full stack application built using HTML5, CSS3, JavaScript and Node.js/Express.js. It includes custom booking and contact forms and utilizes the Facebook and Blogger APIs. Since it is a business website, SEO is important. I used Semrush, Google Analytics, and Google Lighthouse audits to guide SEO.

Team Up - Wanna Start a Game?

If I’m not coding or doing yoga I might be playing Ultimate Frisbee. Once a week we get together for competitive pick-up games. Captains take an attendance list and draft teams at the field. Unfortunately, this can be a rather inefficient use of daylight so I teamed up with my UX designer husband to design and build an app to solve this problem. Team Up takes a list of attendees, each with an ability rating and generates evenly matched teams. Teams are edittable after they’ve been generated in case the captains want to make adjustments. This app is currently still in production. I built it using vanilla Javascript, Node.js/Express.js and NeDB. Then, I conducted user-testing with my husband and frisbee friends. I will be using the information gleaned from the testing to make modifications and rebuild Team Up using React.
